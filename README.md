# WePark

WePark is a MERN-stack community platform designed to bring people together based on their cities and interests. Communities can host events, share announcements, run discussion channels, and even sell merchandise through their own storefronts. The platform is built with scalability in mind, supporting SaaS monetization models such as subscription fees and transaction-based revenue.

---

## 🚀 Features
- **Community Creation & Discovery**: Users can create communities and search/recommend based on city.
- **Events & Meetups**: Communities can post events, manage RSVPs, and send notifications.
- **Community Channels**: Dedicated spaces for posts, announcements, and discussions.
- **Merchandise Storefronts**: Communities can sell their own merchandise with integrated payments.
- **Monetization**: Subscription model for premium features, transaction fees on merchandise sales, and monthly community maintenance fees.
- **Scalable Deployment**: Containerized with Docker, CI/CD pipelines, and cloud hosting support.

---

## 🛠 Tech Stack
- **Frontend**: React, Redux/Context, TailwindCSS or Material-UI
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (multi-tenant design with community/user/event schemas)
- **Authentication**: JWT, OAuth (Google/Facebook optional)
- **Payments**: Stripe/PayPal integration
- **Deployment**: Docker, Nginx reverse proxy, CI/CD (GitHub Actions), Cloud hosting (AWS/GCP/Render)
